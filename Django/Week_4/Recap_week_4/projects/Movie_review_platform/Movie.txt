-----------------------

Distributer can add one or many movies,
Distributer can add one or many genre,
Genre can stay one or many movies,


User can give rating and review to one or many movies,
User can give only one rating and review in one movie,
User can edit and delete review and rating. (when it will be delete then the average rating will be change)



If admin delete Specific Distributer then automatically delete these movies which add by the deleted distributer.
Distributer can delete movies which he/she was add.
Distributer can edit movie which he/she was add.
